How to compile and run the e-commerce Java project:

1. Open a terminal in the project root directory.
2. Compile all Java files:
   javac -d . src/model/Product.java src/service/ProductService.java src/service/OrderService.java src/app/ECommerceApp.java src/Main.java
3. Run the application:
   javac -d . src/model/Product.java src/service/ProductService.java src/service/OrderService.java src/app/ECommerceApp.java src/Main.java
This will start the CLI for adding, listing, and ordering products.
